# RP — Фан-сайт «Реальные пацаны»

![PHP Version](https://img.shields.io/badge/PHP-8.1-blue)
![Version](https://img.shields.io/badge/version-1.0.0-green)

## Описание проекта

RP — веб-приложение, посвящённое телесериалу «Реальные пацаны».

Сайт предоставляет пользователям возможность:

- читать информацию о сериале;
- смотреть галерею;
- просматривать информацию об актёрах;
- смотреть трейлеры;
- оставлять комментарии;
- регистрироваться и входить в систему;
- использовать личный кабинет.

## Установка

1. Клонировать проект:

git clone https://github.com/andrei674427/RP-Boldyrev2026

2. Установить зависимости:

composer install

3. Создать базу данных:

real_patsany

4. Настроить файл .env

5. Запустить OpenServer

6. Открыть:

http://rp/
