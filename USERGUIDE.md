# Руководство пользователя — RP

## Главная страница

После открытия сайта пользователь попадает на главную страницу, где может ознакомиться с информацией о сериале.

## Регистрация

1. Нажать кнопку «Регистрация».
2. Ввести логин.
3. Ввести пароль.
4. Подтвердить регистрацию.

## Авторизация

1. Нажать «Вход».
2. Ввести логин.
3. Ввести пароль.
4. Нажать «Войти».

## Профиль

В профиле пользователь может:

- изменить имя;
- посмотреть свои данные;
- выйти из аккаунта.

## Галерея

Раздел содержит фотографии из сериала.

## Актёры

Раздел содержит информацию об актёрах.

## Трейлеры

Раздел содержит видео и трейлеры.

## Комментарии

Авторизованный пользователь может оставить комментарий.

## Частые вопросы

**Не получается войти**  
→ проверить логин и пароль.

**Не открывается сайт**  
→ проверить OpenServer.

**Ошибка базы данных**  
→ проверить настройки .env.
